Base Wallet's smart wallet mode ("Base mode") is not yet supported on Acurast. If you have it enabled, you will need to disable it before connecting to the Acurast Hub.

## Error

When trying to connect with "Base mode" enabled, you will see the following error:

**Connection Failed** — *Please disable "Base mode" in the Wallet settings and try again.*

## How to disable Base mode

1. Open the Coinbase Wallet app and tap your **profile icon** (top-left).

<ThemedImage
  sources={{
    light: useBaseUrl("/guide/base-wallet/2-go-to-profile.png"),
    dark: useBaseUrl("/guide/base-wallet/2-go-to-profile.png"),
  }}
/>

<div class="padding-vert--md"></div>

2. Scroll down and toggle off **Base mode**.

<ThemedImage
  sources={{
    light: useBaseUrl("/guide/base-wallet/3-disable-base-mode.png"),
    dark: useBaseUrl("/guide/base-wallet/3-disable-base-mode.png"),
  }}
/>

<div class="padding-vert--md"></div>

3. Go back to [hub.acurast.com](https://hub.acurast.com/) and reconnect your wallet.
