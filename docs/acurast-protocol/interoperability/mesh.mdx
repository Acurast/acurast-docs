# Mesh

The **Mesh** interoperability module focuses on **IoT**.

As Acurast is leveraging the **Trusted Execution Environment** of mobile devices, secured access to Bluetooth modules is available natively that can be leverage to securly and verifiably communicate with Bluetooth LE devices such as IoT sensors.
